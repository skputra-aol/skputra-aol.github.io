
###### STEP INSTALL AND USE AWS LOCALSTACK

#### Preparing
- Python pip

#### Install localstack
$ pip install localstack

#### Run localstack
$ DEBUG=1 localstack start
//// Ref : https://github.com/localstack/localstack-workshop/tree/main

#### install aws cli
$ curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
$ aws --version

#### install aws cli - local
$ pip install awscli-local
$ awslocal
///ref : https://github.com/localstack/awscli-local

#### Configuring aws credential
$ aws configure
AWS Access Key ID [****************test]: test
AWS Secret Access Key [****************test]: test
Default region name [us-east-1]: us-east-1
Default output format [None]: 

#### Create S3 bucket NewOne(demo-bucket)
$ aws --endpoint=http://localhost.localstack.cloud:4566 s3 mb s3://demo-bucket
// or ,with simple awslocal
$ awslocal s3 mb s3://demo-bucket2
 
#### Test copy file(README.MD) to S3 bucket(demo-bucket)
$ awslocal s3 cp README.md s3://demo-bucket/README.MD
$ awslocal s3 cp README.md s3://demo-bucket/folder2/README2.MD (auto create folder2 if not exist)

#### Listing all files in S3 bucket(demo-bucket)
$ awslocal s3 ls s3://demo-bucket
$ awslocal s3 ls s3://demo-bucket/folder2

#### For .Net development only
$ sudo apt update
$ sudo apt-get install dotnet-sdk-6.0
